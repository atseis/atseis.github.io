处理静态资源响应

不赘述，尽管 gorilla/mux 确实提供了对静态资源的支持，但是一般还是用特有的工具如 Nginx 处理静态资源，Nginx作为一款强大的反向代理服务器，并发处理静态资源的能力很强，没必要自己去处理这块逻辑

服务单页面应用

对于单页面应用：

  • 首次访问时（后端初始化前端）请求后端入口路由初始化前端资源
  • 后续交互都（前端调后端API）是通过前端代码调用后端 API 接口完成数据渲染
